Emacs Rocks

Einleitung

Wer das Freie Software Magazin liest, hat vielleicht schon einmal von Emacs gehört. Falls nicht, es handelt sich um einen sehr flexiblen Texteditor. So flexibel, dass manche ihr Betriebsystem zum Bootloader degradieren, dass nur dazu da ist Emacs zu starten.

Leider ist die Lernkurve relativ steil was es Neulingen oft erschwert einen Zugang zu finden. Außerdem gibt es viele alternative Editoren und die meisten sind für Einsteiger zugänglicher als Emacs. Aber wie in dieser Artikelreihe gezeigt werden soll, zahlt es auf lange Sicht mehr als aus, die Benutzung von Emacs zu erlernen.

Wieso Emacs?

Besonders für Programmierer gibt es eine Fülle von Angeboten, und oft gibt es Editoren die speziell zur Verwendung einer bestimmten Programmiersprache angeboten werden. Der Nachteil ist, dass man beim Wechsel zu einer anderen Programmiersprache, eventuell einen anderen Editor benutzen muss, da der Support der anderen Sprache in dem verwendeten Editor unter Umständen mangelhaft ist.

Hinzu kommt, dass viele Editoren Tastenkürzel anbieten um bestimmte oft genutzte Funktionen, schneller ausführen zu können ohne sich immer wieder durch die selben Menüs klicken zu müssen. Wenn man Glück hat dann bieten die verschiedenen Editoren die man nutzt eine Anpassung dieser Kürzel an, so dass man in verschiedenen Editoren die gleichen Tastenkürzel verwenden kann. Dies ist jedoch meistens nicht der Fall oder man kann nur bestimmte Funktionen mit Tastenkürzeln belegen.

Weiterhin wäre es doch schön, wenn man dieselben Tastenkürzel die man im Editor verwendet auch beim schreiben von Email oder bei der Eingabe von Befehlen auf der Kommandozeile nutzen könnte.

Ein schöne Eigenschaft von Emacs ist, dass es ein sehr modulare Editor ist und es für fast jede Programmiersprache Pakete gibt. Weiterhin gibt es Erweiterungen mit denen sich ein aus Emacs ein Emailclient, IRC Client und vieles anderes machen lässt. Im Prinzip ist Emacs ein großer REPL der zur Laufzeit durch ausführen von Code beliebig verändert werden kann. Das geht soweit, dass andere Editoren wie z.B. Vi/Vim in Emacs emuliert werden können, falls man deren Funktionalität gerne verwenden möchte.

Emacs bietet ein einheitliches Interface für viele Aufgaben die man am Computer erledigt.

Ehe man sich versieht werden Programme wie der Dateimanager, der Terminal Emulator, Emailclient, IRC CLient, Git Client, Feedreader und mehr durch Emacs ersetzt.

Die erste öffentliche Version von GNU Emacs erschien bereits im Jahr 1985 und es gibt eine große Community, welche die Weiterentwicklung von Emacs vorantreibt. Just am 10. April wurde die Version 24.5 von Emacs veröffentlicht. Man kann getrost davon ausgehen, das der Support die eigene Lebenszeit überdauert, dass heisst das einmal gelernte bleibt mit hoher Wahrscheinlichkeit ein Leben lang von nutzen.

Installation

Aber zunächst einmal zur Installation der neuen Version direkt von der "Quelle", wie es sich für Freie Magazin Leser gehört.

Den Tarball findet man man auf dem FTP der FSF oder wenn man Git benutzt kann man sich das Emacs Repository auch direkt klonen: 

git clone git://git.sv.gnu.org/emacs.git

Falls man sich den Tarball geladen hat, kann man diesen über 

tar xzvf emacs-24.5.tar.gz

entpacken.

Dann noch schnell ein paar Abhängigkeiten installieren. 

# Für Debian-based
sudo apt-get install build-essential
sudo apt-get build-dep emacs24
# Für Fedora/Red Hat-based
sudo yum groupinstall “Development Tools”
sudo yum install yum-utils
sudo yum-builddep emacs

Nach dem Wechsel in das Emacs Verzeichnis muss man dann noch folgende Kommandos ausführen um Emacs zu installieren. 

# Start Manual Installation
./autogen.sh
# Add prefix and path to where you want to install. This is useful if you work on a remote server without root access:
#  You have to use gtk3 to be able to use true type fonts.
./configure
# If ./configure tells you there are dependencies missing, you should install them.
# build it
make
# install it
sudo make install

und schon ist man fertig.

Tipps

Swap Control and Capslock Capslock ist eine selten genutzte Taste. Weil die CTRL Taste in Emacs häufig zum Einsatz kommt, wird oft empfohlen die CAPSLOCK Taste mit der CTRL Taste auszutauschen. Die Anleitung für so ziemlich jede Distribution findet ihr hier. Allerdings auf Englisch.
