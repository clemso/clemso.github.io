Emacs Rocks - Part 1

Was ist Emacs

Wer sich mit Freier Software beschäftigt, hat vielleicht schon einmal von GNU Emacs gehört. Falls nicht, es handelt sich um einen sehr flexiblen Texteditor. So flexibel, dass manche ihr Betriebsystem lediglich als Bootloader betrachten, der nur dazu dient Emacs zu starten. Das mag ein wenig übertrieben sein, aber fest steht, Emacs lässt sich, auf vielfältige Weise für Aufgaben vor dem Bildschirm verwenden. Hier ein paar der geläufigsten Verwendungen von Emacs:

  • Editor
  • Dateimanager
  • Terminal Emulator
  • Email Client
  • IRC Client
  • Feedreader
  • Notizverwaltung
  • Git Interface

Leider ist die Lernkurve relativ steil, was es Neulingen oft erschwert einen Zugang zu finden. Auch ist es schwierig Emacs wirklich effektiv zu nutzen ohne ein wenig über die interne Programmiersprache von Emacs (Elisp) zu lernen. Aber für Menschen die viel mit dem Computern arbeiten, zahlt sich die Mühe auf lange Sicht mehr als aus.

Ein weiterer Vorteil ist, dass dieses Wiseen mit hoher Wahrscheinlichkeit ein Leben lang von Nutzen bleibt, denn die erste öffentliche Version von GNU Emacs erschien bereits im Jahr 1985 und es existiert eine große Community, welche die Weiterentwicklung von Emacs bis zum heutigen Tag aktiv vorantreibt. Es ist nicht abzusehen, dass sich dies in nächster Zeit ändert.

Installation

Den Tarball findet man man auf dem FTP der FSF. Falls man Git benutzt, kann man sich das Emacs Repository auch direkt klonen: 

$ git clone git://git.sv.gnu.org/emacs.git

Falls man sich den Tarball geladen hat, kann man diesen über 

$ tar xzvf emacs-24.5.tar.gz

entpacken.

Dann muss man noch schnell ein paar Abhängigkeiten installieren. Für Debian basierte Systeme geht dies mit 

$ sudo apt-get install build-essential
$ sudo apt-get build-dep emacs24

Für Red-Hat basierte Systeme 

$ sudo yum groupinstall “Development Tools”
$ sudo yum install yum-utils
$ sudo yum-builddep emacs

Nach dem Wechsel in das Emacs Verzeichnis muss man dann noch folgende Kommandos ausführen um Emacs zu installieren. 

$ ./autogen.sh
$ ./configure
$ make
$ sudo make install
